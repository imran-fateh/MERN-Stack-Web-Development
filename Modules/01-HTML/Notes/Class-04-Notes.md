## Semantic Text Tags

## Topics Covered

- mark
- small
- ins
- cite
- dfn
- address

## Important Notes

1. <mark> highlights text.
2. <small> displays smaller text.
3. <ins> shows inserted text.
4. <cite> is used for references and citations.
5. <dfn> is used for defining a term.
6. <address> contains contact information.
7. Inline elements only take required width.
8. Block elements take full width.

## Examples

<mark>Highlighted Text</mark>

<small>Small Text</small>

<ins>Inserted Text</ins>

<cite>HTML Book</cite>

<dfn>HTML</dfn>

<address>
City: Peshawar
</address>
