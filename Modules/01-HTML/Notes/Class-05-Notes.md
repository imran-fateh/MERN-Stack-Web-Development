## Lists and Links

## Topics Covered

- Unordered List (ul)
- Ordered List (ol)
- List Item (li)
- Anchor Tag (a)

## Important Notes

1. ul creates a bulleted list.
2. ol creates a numbered list.
3. li represents a list item.
4. a tag is used to create hyperlinks.
5. href attribute stores the destination URL.

## Examples

<ul>
    <li>Apple</li>
    <li>Orange</li>
</ul>

<ol>
    <li>HTML</li>
    <li>CSS</li>
</ol>

<a href="https://www.google.com">Google</a>
