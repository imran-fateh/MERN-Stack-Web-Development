## Semantic Layout

## Topics Covered

- header
- nav
- main
- section
- article
- id
- Internal Links

## Important Notes

1. header contains introductory content.
2. nav contains navigation links.
3. main contains the main content of the page.
4. section groups related content.
5. article represents independent content.
6. id is used for unique identification.
7. Internal links jump to sections within the page.

## Example

<header>
<nav></nav>
</header>

<main>
<section>
<article></article>
</section>
</main>
