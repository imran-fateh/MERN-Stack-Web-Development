Basic Form Notes

Learning Objective

Learn how to collect basic user information using HTML forms.

---

Topics Covered

- form
- label
- input
- text
- email
- submit

---

Concept Explanation

<form>
Creates a form container.<label>
Provides a label for an input field.<input>
Used to collect user data.type="text"
Accepts normal text input.

type="email"
Accepts email addresses.

type="submit"
Submits the form.

---

Example Structure

<form><label>Name:</label>
<input type="text">

<label>Email:</label>
<input type="email">

<input type="submit">

</form>---

Important Notes

1. form is the parent element.
2. label describes an input field.
3. input collects user data.
4. text accepts normal text.
5. email accepts email addresses.
6. submit sends form data.

---

Advanced Form Notes

Learning Objective

Learn how to build more useful and user-friendly HTML forms.

---

Topics Covered

- password
- textarea
- placeholder
- required
- button

---

Concept Explanation

type="password"
Hides typed characters.

<textarea>
Used for long text input.

placeholder
Displays a hint inside an input field.

required
Makes a field mandatory.

<button>
Creates a clickable button.

---

Example Structure

<form>

    <input
        type="password"
        placeholder="Enter Password"
        required>

    <textarea></textarea><button type="submit">
    Submit

</button>

</form>---

Important Notes

1. Password fields hide user input.
2. Textarea is used for comments and messages.
3. Placeholder provides user guidance.
4. Required fields must be completed.
5. Button is commonly used in professional forms.
6. Forms are used for login, registration and contact pages.
