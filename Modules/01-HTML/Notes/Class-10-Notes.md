Concept Explanation

<iframe>

Embeds another webpage inside the current webpage.

---

src

Specifies the URL to display inside the iframe.

---

width

Sets iframe width.

---

height

Sets iframe height.

---

title

Provides accessibility information.

---

<main>

Represents the main content of a webpage.

---

<section>

Groups related content together.

---

<article>

Represents independent content.

---

<aside>

Contains sidebar or additional information.

---

<datalist>

Provides predefined suggestions for input fields.

---

<input type="file">

Allows users to upload files.

---

Example Structure

<iframe
    src="https://example.com"
    width="600"
    height="300">
</iframe>

<main>

    <section>

        <article>
            Article Content
        </article>

    </section>

    <aside>
        Sidebar Content
    </aside>

</main>

---

Important Notes

1. iframe loads another webpage inside your webpage.
2. Always use title attribute with iframe.
3. main should contain the primary content.
4. section groups related content.
5. article represents standalone content.
6. aside is commonly used for sidebars.
7. datalist provides suggestions while typing.
8. file input is used for uploading files.
9. Semantic tags improve readability and SEO.
