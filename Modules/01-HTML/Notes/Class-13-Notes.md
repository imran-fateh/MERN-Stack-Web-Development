## Important Notes

1. Always use alt attribute with images.

2. Use figure and figcaption for image descriptions.

3. Use controls attribute for audio and video.

4. Always add a favicon to professional websites.

5. Learn HTML entities because they are used frequently.

6. Understand relative paths before starting CSS and JavaScript.

7. Accessibility is an important part of professional web development.

8. Semantic HTML improves SEO and accessibility.
