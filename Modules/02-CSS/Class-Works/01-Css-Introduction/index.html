CSS Fundamentals - Day 1

Today I started my CSS journey and learned the fundamentals of styling web pages. I explored Inline CSS, Internal CSS, and External CSS to understand how styles are applied in real-world websites. This is the first step toward building modern, responsive, and professional web applications.

Inline CSS

Inline CSS is applied directly inside an HTML element using the style attribute.

Internal CSS

Internal CSS is written inside the style tag and is used to style a single web page.

External CSS

External CSS is written in a separate CSS file and linked with the HTML document using the link tag.
