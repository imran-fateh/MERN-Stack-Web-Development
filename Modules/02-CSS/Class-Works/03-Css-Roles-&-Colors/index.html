CSS Rules & Specificity Day - 3

This page demonstrates different CSS Rules, Specificity, Selector Priority, and the !important keyword. These concepts help developers understand how CSS decides which style should be applied when multiple rules target the same element.

What Are CSS Rules?

CSS Rules define how styles are applied to HTML elements.

A CSS Rule contains:

  • Selector
  • Property
  • Value

Last Rule Wins

If multiple CSS rules target the same element and have equal priority, the last rule is applied.

CSS Specificity

Specificity is the priority system used by CSS to determine which style should be applied.

  1. !important
  2. Inline CSS
  3. ID Selector
  4. Class Selector
  5. Element Selector

This heading contains:
