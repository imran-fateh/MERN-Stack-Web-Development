CSS Backgrounds - Day 04

CSS Background Properties are used to control colors, images, positioning, repetition, and attachment of backgrounds.

Background Color

Background Color applies a color behind an HTML element.

Background Color Example

Background Image

Background Image displays an image behind an element.

Background Image Example

Background Repeat

Background Repeat controls whether an image repeats or not.

No Repeat Example

Background Size

Background Size controls the size of a background image.

  • auto
  • contain
  • cover

Background Attachment

Background Attachment controls how the image behaves while scrolling.

  • scroll
  • fixed
  • local

Background Position

Background Position controls the location of a background image.

  • top left
  • top center
  • top right
  • center center
  • bottom left
  • bottom center
  • bottom right
  • 50% 50%

Background Shorthand

Multiple background properties can be written in a single line.

background: url(image.jpg) red no-repeat fixed;
