🎯 CSS Roles

---

📖 What are CSS Rules?

📌 Definition

CSS Rules are used to apply styles to HTML elements.

❓ Why Do We Use CSS Rules?

CSS Rules help control the appearance of a webpage such as:

- 🎨 Colors
- 🔤 Fonts
- 📏 Spacing
- 🖼️ Borders
- 📐 Layouts

📝 Syntax

selector {
property: value;
}

💻 Example

h1 {
color: blue;
}

🔍 Explanation

The "h1" selector targets all heading elements and changes their text color to blue.

---

🔄 Last Rule Wins

📌 Definition

When multiple CSS Rules target the same element with equal priority, the last rule is applied.

💻 Example

h1 {
color: red;
}

h1 {
color: blue;
}

h1 {
color: green;
}

✅ Result

Green Color Applied

🔍 Explanation

All selectors have the same priority, so the browser applies the last rule.

---

🎯 Specificity

📌 Definition

Specificity determines which CSS Rule has higher priority.

💻 Example

<h1 class="heading" id="title">
    CSS Roles
</h1>

h1 {
color: blue;
}

.heading {
color: purple;
}

#title {
color: green;
}

✅ Result

Green Color Applied

🔍 Explanation

The ID Selector has higher priority than the Class Selector and Element Selector.

---

🏷️ CSS Priority Order

📊 Lowest → Highest

Element Selector
↓
Class Selector
↓
ID Selector
↓
!important

---

⚡ !important Rule

📌 Definition

The "!important" keyword gives a CSS property the highest priority.

💻 Example

h1 {
color: wheat !important;
}

🔍 Explanation

The browser applies this property even if other selectors have higher specificity.

---

🎯 Summary

✅ CSS Rules apply styles to HTML elements.

✅ Last Rule Wins when priority is equal.

✅ Specificity decides which rule is applied.

✅ ID Selector > Class Selector > Element Selector.

✅ "!important" has the highest priority.
